// o que é isso

A Galeria GitHub é um espaço onde os membros do Servidor dos Programadores expõem seus perfis do GitHub para a comunidade.

A ideia é simples: conectar devs, mostrar quem está construindo o quê e facilitar o networking entre os membros do servidor.

// como funciona

Os membros vinculam sua conta do GitHub pelo bot do servidor no Discord. As informações — nome, bio, empresa, blog, seguidores e repositórios — são buscadas diretamente da API do GitHub e exibidas aqui em tempo real.

// comunidade
